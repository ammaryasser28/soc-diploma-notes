# Keyloggers

## Overview
A **Keylogger** is a tool (hardware or software) that records every keystroke typed on a keyboard.  
It can capture sensitive information including:
- Usernames and passwords
- Messages and emails
- Banking or personal data

Keyloggers are widely used both maliciously by attackers and legitimately by companies for monitoring.

---

## Uses

### 1. Malicious Use
Attackers use keyloggers to:
- Steal login credentials
- Capture sensitive personal or financial information
- Spy on users
- Facilitate further attacks

### 2. Legitimate Use
Companies may use keyloggers to:
- Monitor employee activity
- Prevent data leakage
- Assess productivity

## How Keyloggers Work
1. User types on the keyboard
2. Keylogger intercepts keystrokes
3. Records them in a local file or sends them remotely
4. Attacker retrieves and uses the data

---

## Risks
- Account compromise
- Identity theft
- Financial fraud
- Privacy violations

---

## Protection Against Keyloggers
1. Use **Antivirus / Anti-malware software**
2. Avoid opening unknown email attachments or links
3. Use **Virtual Keyboards** for sensitive input
4. Enable **Two-Factor Authentication (2FA)**
5. Keep software and operating systems **up to date**
6. Inspect hardware for suspicious devices

---

## Summary
- Keyloggers capture all keyboard input
- Can be used for malicious or legitimate monitoring
- Software keyloggers are the most common today
- Protection relies on **awareness and security tools**
