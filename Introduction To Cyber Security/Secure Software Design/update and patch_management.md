## ðŸ›¡ Patch Management

### What is Patch Management?
- A system to **organize, approve, and deploy updates** efficiently, especially in large organizations
- Prevents network congestion caused by all devices downloading updates independently
- Allows testing updates before widespread deployment to avoid system instability

### Key Points
- Large organizations (1,000+ devices) require **centralized patch management**
- Testing updates ensures stability and reduces risk of crashes
- Helps ensure updates are installed safely and on schedule

## ðŸ”‘ Windows Server Update Services (WSUS)

### What is WSUS?
- WSUS = **Windows Server Update Services**
- Microsoft solution for **centralized patch management**
- Allows administrators to **approve and control updates** for all devices in a network

### How WSUS Works
1. Endpoints **do not download updates directly from Microsoft**
2. WSUS server downloads updates and stores them centrally
3. Administrators approve updates before deployment
4. Only approved updates are sent to endpoints

### Benefits
- Full control over updates
- Reduces risk of unstable systems
- Minimizes network congestion
- Suitable for large organizations
- Ensures tested updates are applied safely
